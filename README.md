Features implemented:
-------------------------------------------------------------------------------

• All required features

• Reflection

• Specular highlights

• Soft shadows and area lights

• Anti-aliasing

• Depth of field

• Interactive camera (A-left, D-right, W-up, S-down, Z-forward, X-backward)

• Refraction (not Fresnel so doesn’t look that good…)
